Sound Design

for this project the goal was to create a radio advert using the free platform software called Audacity. We had to create the advert using our own voice over.

i firstly started by writing my voice over script. I then recorded it using the audacity software. I found using the audacity platform straight yielded the best results for the voiceover. I then applied some techniques to the voice and overall clip to achieve a radio advert.

After getting my voice over satisfactory in length, I then wanted to make my voice deeper i did this by increasing the length of the clip slightly. I then followed a tutorial to furthermore achieve a radio advert sound. I then added sound effects and other techniques to finish the project off. (See below for a layout)

What i Learned: I found it useful to always save your clips when you make sufficent progress. I feel like if something goes wrong It can be hard to simply undo the work sometimes.
